# 🚀 Challenge Management Application

A full-stack web application built using **React.js**, **Spring Boot**, and **H2 Database** that enables users to create and manage monthly challenges dynamically.

## ✨ Features

* 📝 Create and manage monthly challenges with an intuitive user interface.
* 🔄 RESTful APIs powered by Spring Boot.
* 💾 Lightweight and efficient data storage using H2 Database.
* 🧩 Component-based frontend architecture with React.js for enhanced reusability and maintainability.
* 🏗️ Clean separation of concerns following the MVC architectural pattern.
* ⚡ Fast development and testing with an in-memory database.

## 🛠️ Technology Stack

### 🎨 Frontend

* React.js
* Bootstrap
* JavaScript
* HTML5 & CSS3

### ⚙️ Backend

* Spring Boot
* Java
* REST APIs

### 🗄️ Database

* H2 Database

## 🏛️ Architecture

The application follows a layered architecture to ensure scalability, maintainability, and clean code practices:

### 🎯 Controller Layer

Handles incoming HTTP requests and exposes REST API endpoints.

### 🧠 Service Layer

Contains the core business logic and application workflows.

### 📚 Repository Layer

Manages database interactions and data persistence.

## 🌟 Why React?

React's component-based architecture enables:

* ♻️ Reusable UI components
* 📖 Improved code readability
* 🔧 Easier maintenance and scalability
* 🚀 Faster development cycles

## 🎯 Project Goal

This application demonstrates the implementation of a modern full-stack architecture by combining a responsive React frontend with a robust Spring Boot backend, providing a scalable foundation for challenge tracking and management.
